# 🚀 AI Business Automation Suite (Free LLM + Colab)

## 📌 Overview
This project is a multi-module AI automation system built in Google Colab using open-source language models.  
It simulates real-world business AI tools such as email assistants, cover letter generators, travel planners, and chatbots — without requiring paid APIs.

---

## 🎯 Key Features

✉️ Email Reply Generator → Generates professional email responses  
📄 Cover Letter Generator → Creates job-specific cover letters  
✈️ Travel Planner → Builds itineraries based on destination & interests  
💬 AI Chatbot → Conversational business assistant  
📊 Automation Tools → Extensible for Excel, HR, and documents  

---

## 🧠 Tech Stack

- Python  
- Gradio (UI)  
- Hugging Face Transformers  
- PyTorch  
- Google Colab  

---

## 🤖 AI Model Used

- microsoft/DialoGPT-medium
